# chordoma

jazz/funk/electronic chord grid for monome norns + grid

## features

- 120 chord pads (8 rows x 15 cols)
- 5 themes: Jazz Nights, Funk Machine, Electronic, Neo Soul, Modal Drift
- 22 chord types (triads through 13ths)
- complexity tiers by row (simple outer rows, complex center)
- per-row arpeggiator toggles (col 16)
- 7 arp rates
- MIDI out
- 3-page screen (Play, Tempo, Theme)

## controls

- **E1**: page select
- **E2**: BPM (tempo page) / theme select (theme page)
- **E3**: arp rate (tempo page)
- **K2**: panic (all notes off)
- **K3**: randomize chord layout

## grid layout

- **cols 1-15**: chord pads
- **col 16**: arpeggiator toggle per row
- center row (row 4): themed chord progression
- brightness fades from center outward

## requirements

- monome norns
- monome grid
- MIDI connection (optional)

## install

```
;install https://github.com/jamminstein/chordoma
```
